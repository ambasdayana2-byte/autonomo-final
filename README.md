# Juego Piedra, Papel o Tijera con Python

## Descripción del proyecto

Este proyecto consiste en el desarrollo de un juego interactivo de **Piedra, Papel o Tijera**, creado con **Python 3.13** y una interfaz gráfica web usando **HTML, CSS y JavaScript**.

El sistema simula un inicio de sesión básico, muestra un menú principal con tres opciones, permite consultar las reglas del juego, iniciar una partida contra la computadora y finalizar el juego.

La computadora realiza su jugada mediante un algoritmo aleatorio, mientras que el usuario selecciona una de las tres opciones disponibles: piedra, papel o tijera. Luego, el programa compara ambas jugadas y muestra el resultado de la ronda: victoria, derrota o empate.

El proyecto está diseñado con una estructura sencilla, adecuada para un estudiante de nivel inicial en programación. Por esta razón, se aplican conceptos básicos como variables, tipos de datos, operadores, condicionales, estructuras de decisión, funciones y estructuras de datos simples.

---

## Objetivo general

Desarrollar un juego funcional de Piedra, Papel o Tijera utilizando Python como lenguaje principal y tecnologías web para crear una interfaz gráfica sencilla.

---

## Objetivos específicos

- Simular un inicio de sesión básico.
- Crear un menú principal con opciones simples.
- Mostrar las reglas del juego.
- Permitir que el usuario seleccione piedra, papel o tijera.
- Generar una jugada aleatoria para la computadora.
- Comparar la jugada del usuario con la jugada de la computadora.
- Mostrar el resultado de cada ronda.
- Llevar un puntaje de victorias, derrotas y empates.
- Aplicar conceptos básicos de programación vistos en clase.
- Organizar el proyecto en archivos separados para mejorar el orden del código.

---

## Tecnologías utilizadas

- **Python 3.13**: lenguaje principal del proyecto.
- **Flask**: framework utilizado para conectar Python con las páginas HTML.
- **HTML5**: utilizado para crear la estructura de las páginas.
- **CSS3**: utilizado para dar estilos visuales a la interfaz.
- **JavaScript**: utilizado como apoyo básico para la interfaz.
- **Visual Studio Code**: editor de código utilizado para desarrollar el proyecto.
- **Git**: herramienta para control de versiones.
- **GitHub**: plataforma para almacenar el repositorio del proyecto.
